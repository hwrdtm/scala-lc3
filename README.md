# scala-lc3

LC-3 CPU implemented in Scala.

Followed along tutorial https://justinmeiners.github.io/lc3-vm/index.html by @justinmeiners and @rpendleton.

## Instructions

- Run `sbt`
- Run with file `run project/2048.obj`
